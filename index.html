Question 1: How does javascript work?

Answer:
JavaScript is a scripting language and it is an interpreted ,single-threaded,high-level language. JavaScript is considered high-level because it does not require direct interaction with the operating system.Interpreted means the source code is converted to bytecode, machine only understand the bytecode.By using javaScript we can run html and css to connect the script file in html file or write the script tag in html file.JavaScript cannot read all the code at once, an interpreter in the browser reads over the JavaScript code, interprets each line, and runs it.It can set and get the values of cookies, provides messages to the users for information, alert and so on. It also can store temporary data on client-side.JavaScript can modify html pages.

Question 2: Difference between Session storage and Local storage

Answer:
Local storage: Local storage stores a larger amount of data and data store into client computer like key-value pair format.Local storage data don't have expire date.When close the tab local storage data never delete and data can accessible from any window.Local storage capacity about 5MB/10MB. Local storage data are shared between all tabs and windows from the same origin.

Session storage: Session storage stores a larger amount of data and data store into client computer like key-value pair format.Session storage data expire when the tab close.Data can accessible from the same tab only.Session storage capacity about 5MB. If we open multiple tabs/windows with the same URL it creates a sessionStorage for each tab/window.
